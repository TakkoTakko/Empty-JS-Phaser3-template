# Empty Phaser 3 template with JavaScript



## How to use
- Clone this repository
- Run `npm install`
- Run `npm run dev`

###

## How to build
- Run `npm run build`

###

## How to start editing
- change default site structure in `index.html`
- Change style in `style.css`
- Change code in `main.js`
